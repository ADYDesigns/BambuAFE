Bambu Cloud account
The Bambu Cloud requires a User ID and Access token. There are three ways to get this information: Sorting through cookies in your web browser, running a PowerShell script that automates the process, or running a Python script that impersonates a web browser to retrieve the information.

The recommended way is to have PowerShell 7 installed and Download get_bambu_token.ps1 but view the README for full details. The token expires every ~3 months and will need to be renewed.
